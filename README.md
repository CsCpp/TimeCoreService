# 🛡️ TimeCore System Service (MDM Solution)

**TimeCore** — это системное приложение на Kotlin, реализующее функции MDM (Mobile Device Management) и глубокого администрирования для устройств на базе Android (API 24+). Приложение маскируется под обычный конвертер времени, скрывая мощный функционал управления безопасностью.

## ⚠️ ВАЖНО: Подготовка устройства (Обязательно)
Android запрещает назначать «Владельца устройства» (Device Owner), если в системе есть активные пользователи. **Без выполнения этих шагов активация не сработает.**

1. **Удалите ВСЕ аккаунты**: Зайдите в *Настройки -> Пароли и аккаунты* и удалите абсолютно все учетные записи (Google, Telegram, WhatsApp, системные аккаунты производителя и т.д.).
2. **Чистый список**: Убедитесь, что в разделе аккаунтов пусто.
3. **Режим разработчика**: Включите «Отладку по USB» в настройках телефона.

## ⚙️ Активация и проверка через ADB
Выполните эти команды в терминале (CMD или PowerShell) после установки APK:

### 1. Назначение владельцем устройства

adb shell dpm set-device-owner com.android.system.timecore/.AdminReceiver
Успех: Success: Device owner set to package...
Ошибка: Если аккаунты не удалены, вы получите IllegalStateException.

### 2. Выдача прав на управление защищенными настройками (DNS/Интернет)
adb shell pm grant com.android.system.timecore android.permission.WRITE_SECURE_SETTINGS
### 3. Диагностика и проверка статуса

Проверка прав администратора (ищите DeviceOwner)
adb shell dpm list-owners

Проверка разрешения на управление интернетом (ищите granted=true)
adb shell dumpsys package com.android.system.timecore | findstr WRITE_SECURE_SETTINGS

# 🚩 Первый запуск и инициализация
После активации через ADB откройте приложение для первичной настройки:

Master Password: Задайте секретный код (ключ). Он будет хэширован (SHA-256) и станет единственным способом доступа к функциям.

Настройка DNS: Укажите адрес сервера (для разблокировки трафика).

Контроль USB: Включите блокировку передачи данных (USB Data Transfer) для защиты от несанкционированного доступа.

# 🔑 Управление через маскировку (Секретные коды)
Функции MDM активируются через ввод Мастер-пароля в поле конвертера вместо чисел:

Пароль (без суффиксов) — Full Recovery: Мгновенный сброс всех ограничений. Включает видимость скрытых приложений, разрешает работу USB и восстанавливает стандартный интернет.

Пароль + set — Меню администратора: Вход в скрытые настройки. Здесь можно вручную изменить адрес DNS или состояние USB-порта.

Пароль + dns — Network Fix: Быстрая разблокировка только интернета без изменения других политик безопасности.

Пароль + del — Полное удаление: Снятие прав Device Owner и деактивация администратора. После этого приложение можно удалить обычным способом.
